# NAME

**darwin_ramfs_mount** - Create and mount a RAM disk on macOS.

# SYNOPSIS

**`darwin_ramfs_mount`** _directory_ _size_

# DESCRIPTION

Create and mount a RAM disk on macOS.

# OPTIONS

_directory_

:   Directory where the RAM disk is mounted to.

_size_

:   Size of the disk image in MB.
