# NAME

**mj** - Jiggle the mouse.

# SYNOPSIS

**`mj`**

# DESCRIPTION

Jiggle the mouse in case you don't want your computer to go in sleep mode.

Requires at least Java v17 and JBang.

Run `j! app install ~/.local/bin/mj.java` to install.

# OPTIONS

None

# EXAMPLES

    $ mj
